struct vrt_header¶
#include <vrt_types.h>

Header data.

Public Members

vrt_packet_type packet_type¶

Type of packet.

vrt_header_indicators has¶

Indicates presence of extra fields.

vrt_tsm tsm¶

Timestamp mode. Whether timestamps are with fine or coarse resolution. Coarse if an event happened sometime in the data sampling interval.

Note

True is only valid for context packets.

vrt_tsi tsi¶

Type of integer second timestamp.

Note

Rule 6.1.1-7: All the packets in an IF Data Packet Stream shall have the same TSI code, and thus the same Integer-seconds Timestamp.

vrt_tsf tsf¶

Type of fractional second timestamp.

Note

Rule 6.1.1-10: All the packets in an IF Data Packet Stream shall have the same TSF code, and thus the same Fractional-seconds Timestamp.

uint8_t packet_count¶

Modulo-16 count of packets for the specified stream. Can (to some degree) be used to determine packet loss.

Note

Only the 4 least significant bits are used.

Note

Rule 6.1.1-11: The Packet Count shall increment in each consecutive IF Data packet having the same Stream Identifier and the same packet type. The count shall roll over from “1111” to “0000”.

uint16_t packet_size¶

Total number of 32-bit words in the packet.

Note

May vary from packet to packet.
