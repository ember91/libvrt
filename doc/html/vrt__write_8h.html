int32_t vrt_write_context ( const vrt_context *  context,
void *  buf,
uint32_t  buf_words 
)

Low-level function that writes VRT context section.

Parameters
contextContext to write.
bufBuffer to write to. This must point to the first context section word int the packet, i.e. directly following the fields section.
buf_wordsSize of buf in 32-bit words.
Returns
Number of written 32-bit words, or VRT_ERR if error.
Warning
This does not do any validation, i.e. it doesn't check if the field values make any sense.
Fields represented as double but with an underlying 64-bit fixed precision format, i.e. bandwidth, if_reference_frequency, rf_reference_frequency, rf_reference_frequency_offset, if_band_offset, and sample_rate, may in rare cases lose precision since double only has 53 bits of precision.
