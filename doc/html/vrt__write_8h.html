Low-level function that writes VRT header section.

Parameters
headerHeader to write.
bufBuffer to write to.
buf_wordsSize of buf in 32-bit words.
validateTrue if validation should be done. If false, only buffer size is validated.
Returns
Number of written 32-bit words (always 1), or a negative number if error.
Return values
VRT_ERR_BUF_SIZEBuffer is too small.
VRT_ERR_PACKET_TYPEPacket type is an invalid value.
VRT_ERR_TSMData packet has TSM bit set.
VRT_ERR_TSITSI is an invalid value.
VRT_ERR_TSFTSF is an invalid value.
VRT_ERR_PACKET_COUNTPacket count is outside valid bounds (> 0x0F).
Note
May requires output buffer data to be byte swapped if platform endianess isn't big endian (network order).
