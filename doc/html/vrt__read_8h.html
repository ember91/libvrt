Low-level function that reads VRT header section.

Parameters
bufBuffer to read from.
buf_wordsSize of buf in 32-bit words.
headerHeader to read into.
validateTrue if validation should be done. If this is false, only buffer size is validated.
Returns
Number of read 32-bit words, or a negative number if error.
Return values
VRT_ERR_BUF_SIZEBuffer is too small.
VRT_ERR_PACKET_TYPEPacket type is an invalid value.
VRT_ERR_TRAILERContext packet has trailer bit set.
VRT_ERR_TSMData packet has TSM bit set.
VRT_ERR_RESERVEDReserved bits are set.
Note
Requires input buffer data to be byte swapped if platform endianess isn't big endian (network order).

Low-level function that reads VRT IF context section.

Parameters
bufBuffer to read from.
buf_wordsSize of buf in 32-bit words. Starting from the header word.
if_contextIF context struct to read into.
validateTrue if validation should be done. If this is false, only buffer size is validated.
Returns
Number of read 32-bit words, or a negative number if error.
Return values
VRT_ERR_BUF_SIZEBuffer is too small.
VRT_ERR_BANDWIDTHBandwidth is negative.
VRT_ERR_SAMPLE_RATESample rate is negative.
VRT_ERR_TEMPERATURETemperature is below absolute zero (< -273.15).
VRT_ERR_REAL_OR_COMPLEXReal/Complex is an invalid value.
VRT_ERR_DATA_ITEM_FORMATData item format is an invalid value.
VRT_ERR_INTEGER_SECOND_TIMESTAMPInteger second timestamp is not 0xFFFFFFFF when TSI is VRT_TSI_UNDEFINED.
VRT_ERR_FRACTIONAL_SECOND_TIMESTAMPFractional second timestamp is not 0xFFFFFFFFFFFFFFFF when TSF is VRT_TSF_UNDEFINED.
VRT_ERR_REAL_TIMEFractional seconds Real time is active but picoseconds is outside bounds.
VRT_ERR_LATITUDELatitude is outside valid bounds (< -90 or > 90).
VRT_ERR_LONGITUDELongitude is outside valid bounds (< -180 or > 180).
VRT_ERR_HEADING_ANGLEHeading angle outside valid bounds (< 0 or > 359.999999761582).
VRT_ERR_TRACK_ANGLETrack angle is outside valid bounds (< 0 or > 359.999999761582).
VRT_ERR_MAGNETIC_VARIATIONMagnetic variation is outside valid bounds (< -180 or > 180).
VRT_ERR_RESERVEDA reserved field was written to.
Warning
Fields represented as double but with an underlying 64-bit fixed precision format, i.e. bandwidth, if_reference_frequency, rf_reference_frequency, rf_reference_frequency_offset, if_band_offset, and sample_rate, may in rare cases lose precision since double only has 53 bits of precision.
