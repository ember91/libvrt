struct vrt_gps_ascii¶

GPS device ASCII.

Public Members

uint32_t oui¶

IEEE Organizationally Unique Identifier of GPS/INS manufacturer.

Note

Only the 24 least significant bits are used.

uint32_t number_of_words¶

Number of 32-bit words used for the GPS ASCII.

const char *ascii¶

GPS ASCII sentences. Padded with null characters so it’s a multiple of 4 B. Pointer may be null when not defined.

Warning

Will point into the buffer when the field is read. Deallocating the buffer will render the pointer invalid.
