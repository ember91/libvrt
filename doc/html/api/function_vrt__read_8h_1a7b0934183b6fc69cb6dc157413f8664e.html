Function vrt_read_if_context¶

  • Defined in File vrt_read.h

Function Documentation¶

VRT_WARN_UNUSED int32_t vrt_read_if_context (const void *buf, int32_t words_buf, vrt_if_context *if_context, bool validate)

Low-level function that reads VRT IF context section.

Return

Number of read 32-bit words, or a negative number if error.

Warning

Fields represented as double but with an underlying 64-bit fixed precision format, i.e. bandwidth, if_reference_frequency, rf_reference_frequency, rf_reference_frequency_offset, if_band_offset, and sample_rate, may in rare cases lose precision since double only has 53 bits of precision.

Parameters

  • buf: Buffer to read from.

  • words_buf: Size of buf in 32-bit words.

  • if_context: IF context struct to read into.

  • validate: True if validation should be done. If false, only buffer size is validated.

Return Value

  • VRT_ERR_BUFFER_SIZE: Buffer is too small.

  • VRT_ERR_RESERVED: One or multiple reserved bits are set.

  • VRT_ERR_BOUNDS_BANDWIDTH: Bandwidth is outside valid bounds (< 0 Hz).

  • VRT_ERR_GAIN_STAGE2_SET: Gain stage 1 must be used instead of stage 2 when only one is set.

  • VRT_ERR_BOUNDS_SAMPLE_RATE: Sample rate is outside valid bounds (< 0 Hz).

  • VRT_ERR_BOUNDS_TEMPERATURE: Temperature is outside valid bounds (< -273.15 degrees).

  • VRT_ERR_INVALID_REAL_OR_COMPLEX: Real/Complex is an invalid value.

  • VRT_ERR_INVALID_DATA_ITEM_FORMAT: Data item format is an invalid value.

  • VRT_ERR_SET_INTEGER_SECOND_TIMESTAMP: Integer second timestamp is not 0xFFFFFFFF when TSI is VRT_TSI_UNDEFINED.

  • VRT_ERR_SET_FRACTIONAL_SECOND_TIMESTAMP: Fractional second timestamp is not 0xFFFFFFFFFFFFFFFF when TSF is VRT_TSF_UNDEFINED.

  • VRT_ERR_BOUNDS_REAL_TIME: TSF is VRT_TSF_REAL TIME but picoseconds is outside valid bounds (> 999999999999 ps).

  • VRT_ERR_BOUNDS_LATITUDE: Latitude is outside valid bounds (< -90 or > 90 degrees).

  • VRT_ERR_BOUNDS_LONGITUDE: Longitude is outside valid bounds (< -180 or > 180 degrees).

  • VRT_ERR_BOUNDS_HEADING_ANGLE: Heading angle outside valid bounds (< 0 or > ~360 degrees).

  • VRT_ERR_BOUNDS_TRACK_ANGLE: Track angle is outside valid bounds (< 0 or > ~360 degrees).

  • VRT_ERR_BOUNDS_MAGNETIC_VARIATION: Magnetic variation is outside valid bounds (< -180 or > 180 degrees).
