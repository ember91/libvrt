Function vrt_read_trailer¶

  • Defined in File vrt_read.h

Function Documentation¶

VRT_WARN_UNUSED int32_t vrt_read_trailer (const void *buf, int32_t words_buf, struct vrt_trailer *trailer)

Low-level function that reads VRT trailer section.

Return

Number of read 32-bit words, i.e. 1, or a negative number if error.

Note

Unlike other read functions, there is nothing to validate, so no ‘validate’ parameter here.

Parameters

  • buf: Buffer to read from. This must point to the position of the trailer word, i.e. the last word in the packet.

  • words_buf: Size of buf in 32-bit words.

  • trailer: Trailer to read into.

Return Value

  • VRT_ERR_BUFFER_SIZE: Buffer is too small.
