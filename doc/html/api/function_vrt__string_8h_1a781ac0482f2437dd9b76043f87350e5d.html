const char *vrt_string_error(int32_t error)¶

Get a string representation of an error code.

Note

Inputting a code that isn’t an error, i.e. >= 0, returns “Success”. Inputting an unknown error code returns “Unknown”.

Parameters

  • error – Error code.

Returns

String representation.
