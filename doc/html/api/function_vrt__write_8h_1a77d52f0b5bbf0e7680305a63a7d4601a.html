Function vrt_write_packet¶

  • Defined in File vrt_write.h

Function Documentation¶

VRT_WARN_UNUSED int32_t vrt_write_packet (const vrt_packet *packet, void *buf, int32_t words_buf, bool validate)

Higher-level function that writes a full VRT packet.

Return

Number of written 32-bit words, or a negative number if error.

Note

Calculates packet_size field in header to match the number of written words. There is no need to calculate it beforehand. It is not modified in the const packet parameter, though.

Note

May require output buffer data to be byte swapped if platform endianess isn’t big endian (network order).

Note

Will copy body data. For a zero-copy write, use the low-level routines.

Parameters

  • packet: Packet to write.

  • buf: Buffer to write to.

  • words_buf: Size of buf in 32-bit words.

  • validate: True if validation should be done. If false, only buffer size is validated.

Return Value

  • VRT_ERR_BUFFER_SIZE: Buffer is too small.

  • VRT_ERR_INVALID_PACKET_TYPE: Packet type is an invalid value.

  • VRT_ERR_INVALID_TSM: TSM is an invalid value.

  • VRT_ERR_TSM_IN_DATA: Data packet has TSM bit set.

  • VRT_ERR_INVALID_TSI: TSI is an invalid value.

  • VRT_ERR_INVALID_TSF: TSF is an invalid value.

  • VRT_ERR_BOUNDS_PACKET_COUNT: Packet count is outside valid bounds (> 0x0F).

  • VRT_ERR_BOUNDS_REAL_TIME: TSF is VRT_TSF_REAL TIME but picoseconds is outside valid bounds (> 999999999999 ps).

  • VRT_ERR_BOUNDS_OUI: OUI is outside valid bounds (> 0x00FFFFFF).

  • VRT_ERR_BOUNDS_ASSOCIATED_CONTEXT_PACKET_COUNT: Associated context packet count is outside valid bounds (>0x7F).

  • VRT_ERR_BOUNDS_BANDWIDTH: Bandwidth is outside valid bounds (< 0 Hz or > 8.79 THz).

  • VRT_ERR_BOUNDS_IF_REFERENCE_FREQUENCY: IF reference frequency is outside valid bounds (< ~-8.79 or > ~8.79 THz).

  • VRT_ERR_BOUNDS_RF_REFERENCE_FREQUENCY: RF reference frequency is outside valid bounds (< ~-8.79 or > ~8.79 THz).

  • VRT_ERR_BOUNDS_RF_REFERENCE_FREQUENCY_OFFSET: RF reference frequency offset is outside valid bounds (< ~-8.79 or > ~8.79 THz).

  • VRT_ERR_BOUNDS_IF_BAND_OFFSET: IF band offset is outside valid bounds (< ~-8.79 or > ~8.79 THz).

  • VRT_ERR_BOUNDS_REFERENCE_LEVEL: Reference level is outside valid bounds (< -256 or > ~256 dBm).

  • VRT_ERR_BOUNDS_GAIN: Gain stage 1 or 2 outside valid bounds (< -256 or > ~256 dB).

  • VRT_ERR_GAIN_STAGE2_SET: Gain stage 1 must be used instead of stage 2 when only one is set.

  • VRT_ERR_BOUNDS_SAMPLE_RATE: Sample rate is outside valid bounds (< 0 Hz or > ~8.79 THz).

  • VRT_ERR_BOUNDS_TEMPERATURE: Temperature is outside valid bounds (< -273.15 or > ~512 degrees C).

  • VRT_ERR_INVALID_PACKING_METHOD: Packing method is an invalid value.

  • VRT_ERR_INVALID_REAL_OR_COMPLEX: Real/Complex is an invalid value.

  • VRT_ERR_INVALID_DATA_ITEM_FORMAT: Data item format is an invalid value.

  • VRT_ERR_BOUNDS_EVENT_TAG_SIZE: Event tag size is outside valid bounds (> 0x07).

  • VRT_ERR_BOUNDS_CHANNEL_TAG_SIZE: Channel tag size is outside valid bounds (> 0x0F).

  • VRT_ERR_BOUNDS_ITEM_PACKING_FIELD_SIZE: Item packing fields size is outside valid bounds (> 0x3F).

  • VRT_ERR_BOUNDS_DATA_ITEM_SIZE: Data item size is outside valid bounds (> 0x3F).

  • VRT_ERR_SET_INTEGER_SECOND_TIMESTAMP: Integer second timestamp is not 0xFFFFFFFF when TSI is VRT_TSI_UNDEFINED.

  • VRT_ERR_SET_FRACTIONAL_SECOND_TIMESTAMP: Integer second timestamp is not 0xFFFFFFFF when TSI is VRT_TSI_UNDEFINED.

  • VRT_ERR_BOUNDS_LATITUDE: Latitude is outside valid bounds (< -90 or > 90 degrees).

  • VRT_ERR_BOUNDS_LONGITUDE: Longitude is outside valid bounds (< -180 or > 180 degrees).

  • VRT_ERR_BOUNDS_ALTITUDE: Altitude is outside valid bounds (< ~-67108 or > ~67108 km).

  • VRT_ERR_BOUNDS_SPEED_OVER_GROUND: Speed over ground is outside valid bounds (< 0 or > ~65536 m/s).

  • VRT_ERR_BOUNDS_HEADING_ANGLE: Heading angle outside valid bounds (< 0 or > ~360 degrees).

  • VRT_ERR_BOUNDS_TRACK_ANGLE: Track angle is outside valid bounds (< 0 or > ~360 degrees).

  • VRT_ERR_BOUNDS_MAGNETIC_VARIATION: Magnetic variation is outside valid bounds (< -180 or > 180 degrees).

  • VRT_ERR_BOUNDS_POSITION: Position x, y, or z is outside valid bounds (< ~-67108 or > ~67108 km).

  • VRT_ERR_BOUNDS_ATTITUDE: Attitude alpha, beta, or gamma is outside valid bounds (< -512 or > ~512 degrees).

  • VRT_ERR_BOUNDS_VELOCITY: Velocity dx, dy, or dz is outside valid bounds (< -32768 or > ~32768 m/s).

  • VRT_ERR_BOUNDS_SOURCE_LIST_SIZE: Source list size is outside valid bounds (> 0x01FF).

  • VRT_ERR_BOUNDS_SYSTEM_LIST_SIZE: System list size is outside valid bounds (> 0x01FF).

  • VRT_ERR_BOUNDS_CHANNEL_LIST_SIZE: Channel list size is outside valid bounds (> 0x7FFF).

  • VRT_ERR_BOUNDS_PACKET_SIZE: Calculated packet size is outside valid bounds (> 0xFFFF).
