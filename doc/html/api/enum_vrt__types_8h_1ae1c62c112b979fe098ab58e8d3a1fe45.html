enum vrt_tsf¶

Reference-point time in subsecond resolution.

Note

Rule 6.1.1-10: All the packets in an IF Data Packet Stream shall have the same TSF code, and thus the same Fractional-seconds Timestamp.

Values:

enumerator VRT_TSF_NONE¶

No subsecond time resolution.

enumerator VRT_TSF_UNDEFINED¶

Replaces NONE in the Formatted GPS/INS geolocation fields.

enumerator VRT_TSF_SAMPLE_COUNT¶

Number of samples since last second shift.

enumerator VRT_TSF_REAL_TIME¶

Picoseconds since last second shift.

enumerator VRT_TSF_FREE_RUNNING_COUNT¶

Counts samples from the first sample. Will eventually roll over.
