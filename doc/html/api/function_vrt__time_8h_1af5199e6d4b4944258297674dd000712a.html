Function vrt_time_difference¶

  • Defined in File vrt_time.h

Function Documentation¶

VRT_WARN_UNUSED int vrt_time_difference(const struct vrt_packet *p2, const struct vrt_packet *p1, double sample_rate, struct vrt_time *diff)¶

Calculate positive or negative time difference between two packets in whole and fractional seconds.

Note

Ignores any IF context sample rate field in packet. Use sample_rate parameter instead.

Note

If both TSI and TSF are NONE, it is regarded as no time difference.

Note

Time difference is positive if p2 happens after p1.

Return

0, or error code if error.

Parameters

  • p2: Packet 2.

  • p1: Packet 1.

  • sample_rate: Sample rate [Hz].

  • diff: Time difference [out].

Return Value

  • VRT_ERR_MISMATCH_TIME_TYPES: TSI and/or TSF timestamps differ between packets.

  • VRT_ERR_MISSING_SAMPLE_RATE: Sample rate is required but is not provided (<= 0).

  • VRT_ERR_BOUNDS_SAMPLE_COUNT: Fractional timestamp is outside valid bounds (>= sample rate).

  • VRT_ERR_BOUNDS_REAL_TIME: TSF is VRT_TSF_REAL_TIME but picoseconds is outside valid bounds (> 999999999999 ps).

  • VRT_ERR_INTEGER_SECONDS_MISMATCH: Timestamp integer seconds and calculated seconds from the Free running count fractional part differ.
