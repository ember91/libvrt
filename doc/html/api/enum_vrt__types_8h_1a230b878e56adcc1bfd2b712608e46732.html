enum vrt_tsi¶

Reference-point time in resolution of a second.

Values:

enumerator VRT_TSI_NONE¶

No one-second resolution time.

enumerator VRT_TSI_UNDEFINED¶

Replaces NONE in the Formatted GPS/INS geolocation fields.

enumerator VRT_TSI_UTC¶

Seconds since January 1, 1970, GMT.

enumerator VRT_TSI_GPS¶

Seconds since January 6, 1980.

enumerator VRT_TSI_OTHER¶

Some other time reference.
