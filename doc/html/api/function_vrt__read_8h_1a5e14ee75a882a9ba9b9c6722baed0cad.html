VRT_WARN_UNUSED int32_t vrt_read_header(const void *buf, int32_t words_buf, struct vrt_header *header, bool validate)¶

Low-level function that reads VRT header section.

Return

Number of read 32-bit words, or a negative number if error.

Note

Requires input buffer data to be byte swapped if platform endianess isn’t big endian (network order).

Parameters

  • buf: Buffer to read from.

  • words_buf: Size of buf in 32-bit words.

  • header: Header to read into.

  • validate: True if validation shall be done. If false, only buffer size is validated.

Return Value

  • VRT_ERR_BUFFER_SIZE: Buffer is too small.

  • VRT_ERR_RESERVED: One or multiple reserved bits are set.

  • VRT_ERR_INVALID_PACKET_TYPE: Packet type is an invalid value.

  • VRT_ERR_TRAILER_IN_CONTEXT: Context packet has trailer bit set.

  • VRT_ERR_TSM_IN_DATA: Data packet has TSM bit set.
