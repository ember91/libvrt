Function vrt_write_header¶

  • Defined in File vrt_write.h

Function Documentation¶

VRT_WARN_UNUSED int32_t vrt_write_header (const vrt_header *header, void *buf, uint32_t words_buf, bool validate)

Low-level function that writes VRT header section.

Return

Number of written 32-bit words (always 1), or a negative number if error.

Note

May requires output buffer data to be byte swapped if platform endianess isn’t big endian (network order).

Parameters

  • header: Header to write.

  • buf: Buffer to write to.

  • words_buf: Size of buf in 32-bit words.

  • validate: True if validation should be done. If false, only buffer size is validated.

Return Value

  • VRT_ERR_BUF_SIZE: Buffer is too small.

  • VRT_ERR_INVALID_PACKET_TYPE: Packet type is an invalid value.

  • VRT_ERR_INVALID_TSM: TSM is an invalid value.

  • VRT_ERR_TSM_IN_DATA: Data packet has TSM bit set.

  • VRT_ERR_INVALID_TSI: TSI is an invalid value.

  • VRT_ERR_INVALID_TSF: TSF is an invalid value.

  • VRT_ERR_BOUNDS_PACKET_COUNT: Packet count is outside valid bounds (> 0x0F).
